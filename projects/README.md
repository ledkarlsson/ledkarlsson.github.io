# Projects

Each tool gets its own folder here.

- `kartgenerator/` - map generator project page and future project assets.

The code is generated by AI using test-driven development. Tests are ran with Playwright, which checks all logic. The AI runs all tests for every code change, and adjusts accordingly. Every time a human tester finds a bug, the AI is requested to reproduce it with a test, and then solve the bug.

Before this strategy, regression bugs was quite common.
